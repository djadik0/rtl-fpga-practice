# Задача. BCD-счётчик

## Цель работы

Реализовать модуль `binary_coded` на `SystemVerilog` и проверить его работу в моделировании.

## Краткое описание

В проекте реализован 4-битный BCD-счётчик с входами тактирования, сброса, разрешения счёта и параллельной загрузки.

Модуль выполняет счёт от `0` до `9`. При достижении значения `1001` следующий инкремент переводит счётчик обратно в `0`.

Также в модуле формируется сигнал `TC`, который показывает достижение терминального состояния.

## Что реализовано

В модуле `binary_coded` реализованы:

- асинхронный сброс по сигналу `rst`
- параллельная загрузка значения по сигналу `L`
- инкремент счётчика по сигналу `CE`
- ограничение диапазона счёта значениями `0...9`
- формирование выходов `Q0`, `Q1`, `Q2`, `Q3`
- формирование сигнала терминального счёта `TC`

### Особенности работы

- при `rst = 1` счётчик сбрасывается в `0`
- при `L = 1` в счётчик загружается значение `{D3, D2, D1, D0}`
- при `CE = 1` счётчик увеличивается на `1`
- при достижении `1001` следующий счёт переводит значение в `0000`
- сигнал `TC` активен, когда значение счётчика равно `1001`

## Что проверялось в testbench

В testbench проверялись следующие сценарии:

- начальная инициализация сигналов
- асинхронный сброс счётчика
- параллельная загрузка значения `0101`
- последующий счёт при `CE = 1`
- остановка счёта при `CE = 0`
- параллельная загрузка значения `1001`
- проверка сигнала `TC`
- переход счётчика из `1001` в `0000`

Моделирование выполняется с простым генератором тактового сигнала и последовательной подачей управляющих сигналов.

## Результат работы

В результате был получен BCD-счётчик с поддержкой сброса, разрешения счёта и параллельной загрузки.

Работа модуля была проверена в testbench на нескольких сценариях, включая загрузку значений, счёт и переход через граничное значение `9`.

## Вывод

В ходе выполнения задачи был реализован и проверен модуль `binary_coded` на `SystemVerilog`.  
Моделирование показало корректную работу сброса, загрузки, счёта в диапазоне `0...9` и формирование сигнала `TC` в терминальном состоянии.
