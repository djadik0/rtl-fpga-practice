# Задача. Формирователь импульса

## Цель работы

Реализовать модуль `waveform` на `SystemVerilog` и проверить его работу в моделировании.

## Краткое описание

В проекте реализован модуль `waveform`, который по входному сигналу `input_signal` формирует выходной импульс фиксированной длины на выходе `output_signal`.

При появлении активного входного сигнала модуль запускает внутренний процесс формирования импульса. После этого выход удерживается в активном состоянии несколько тактов, а затем автоматически сбрасывается.

## Что реализовано

В модуле `waveform` реализованы:

- синхронная работа по положительному фронту сигнала `clock`
- запуск формирования импульса по сигналу `input_signal`
- внутренний 2-битный счётчик `cnt`
- внутренний сигнал разрешения `en`
- автоматическое завершение импульса после фиксированного числа тактов

### Особенности реализации

Логика работы модуля следующая:

- при `input_signal = 1` выход `output_signal` устанавливается в `1`
- одновременно активируется внутренний сигнал `en`
- счётчик `cnt` сбрасывается в `0`
- пока `en = 1`, счётчик увеличивается на каждом такте
- при достижении `cnt == 2` выходной сигнал сбрасывается в `0`, а формирование импульса завершается

Таким образом модуль формирует выходной импульс длительностью в несколько тактов после входного запуска.

## Что проверялось в testbench

В testbench проверялись следующие сценарии:

- генерация тактового сигнала
- начальное состояние входного сигнала
- первая активация `input_signal`
- повторная активация после паузы
- ещё одна активация входного сигнала
- завершение моделирования после нескольких запусков формирователя

В testbench входной сигнал несколько раз кратковременно устанавливается в `1`, после чего возвращается в `0`.

Это позволяет проверить повторный запуск формирователя импульса и поведение выхода `output_signal` на waveform.

## Результат работы

В результате был получен модуль `waveform`, формирующий выходной импульс фиксированной длительности после подачи входного сигнала запуска.

Работа модуля была проверена в testbench на нескольких отдельных импульсах `input_signal`.

В ходе выполнения задачи был реализован и проверен модуль `waveform` на `SystemVerilog`.  
Моделирование показало корректный запуск формирователя по входному сигналу и автоматическое завершение выходного импульса после заданного числа тактов.
