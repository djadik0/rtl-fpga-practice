# Задача. Flatten layer

## Цель работы

Реализовать модуль `flatten_layer` на `SystemVerilog` и проверить его работу в моделировании.

## Краткое описание

В проекте реализован модуль `flatten_layer`, который преобразует многомерный массив после слоя pooling в одномерный вектор.

На вход модуля подаётся массив `pooling_matrix_o`, содержащий данные по нескольким фильтрам.  
На выходе формируется вектор `flatten_vector_o`, в который элементы последовательно записываются в порядке обхода:

- по фильтрам
- по строкам
- по столбцам

## Что реализовано

В модуле `flatten_layer` реализованы:

- параметризуемая структура входного массива
- параметризуемый размер выходного вектора
- комбиниционное преобразование 3D-массива в 1D-вектор
- последовательная укладка элементов входного массива в выходной вектор

### Особенности реализации

Модуль использует вложенные циклы:

- по номеру фильтра `f`
- по координате строки `y`
- по координате столбца `x`

Текущий индекс выходного вектора хранится во внутренней переменной `t`.

Каждый элемент входного массива `pooling_matrix_o[f][y][x]` последовательно записывается в `flatten_vector_o[t]`.

## Результат работы

В результате был получен параметризуемый модуль `flatten_layer`, выполняющий преобразование выходных данных pooling-слоя из многомерного массива в одномерный вектор.

Работа модуля была проверена в простом testbench с заполнением входного массива тестовыми значениями.

## Вывод

В ходе выполнения задачи был реализован и проверен модуль `flatten_layer` на `SystemVerilog`.  
Моделирование показало корректное преобразование многомерного массива в одномерный вектор и правильный порядок записи элементов на выходе.
