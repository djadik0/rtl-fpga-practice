# SPI Master

## Цель работы

Реализовать и проверить модуль `spi_master` на `SystemVerilog`.

## Краткое описание

В проекте реализован простой SPI Master, который передаёт байт данных по линии `mosi` и принимает байт данных по линии `miso`.

Модуль формирует основные SPI-сигналы:

- `sclk` — тактовый сигнал SPI;
- `mosi` — данные от master к slave;
- `miso` — данные от slave к master;
- `cs_n` — активный низким уровнем сигнал выбора slave;
- `busy` — модуль занят передачей;
- `done` — передача завершена.

## Что проверялось в testbench

В testbench проверялась передача данных через SPI-интерфейс:

- подача сигнала `start`;
- загрузка значения `tx_data`;
- формирование `cs_n`;
- генерация `sclk`;
- последовательная передача битов по `mosi`;
- приём битов с линии `miso`;
- завершение передачи по сигналу `done`.

## Waveform

На waveform видно, что после подачи `start` модуль активирует `cs_n`, формирует тактовый сигнал `sclk`, передаёт данные по `mosi`, принимает данные по `miso`, а после завершения обмена выставляет `done`.
